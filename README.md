# Google Search AI Agent

An AI-powered Google Search Assistant built using Python that helps users perform intelligent web searches and retrieve relevant information efficiently.

## Features

* AI-powered search assistance
* Google search integration
* Fast and intelligent query handling
* Clean Python project structure
* Environment variable support using `.env`
* Easy dependency management with `requirements.txt`

---

## Project Structure

```bash
Google_search_ai_aagent/
│
├── main.py                # Main application file
├── Project.ipynb          # Jupyter notebook for testing/experiments
├── requirements.txt       # Python dependencies
├── pyproject.toml         # Project configuration
├── .gitignore             # Ignored files/folders
├── .env                   # Environment variables (not uploaded)
└── README.md              # Project documentation
```

## Installation

### 1. Clone Repository

```bash
git clone https://github.com/rhtmishra19/Google-search-Ai-Agent.git
```

### 2. Navigate to Project Folder

```bash
cd Google-search-Ai-Agent
```

### 3. Create Virtual Environment

```bash
python -m venv .venv
```

### 4. Activate Virtual Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Mac/Linux

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Environment Variables

Create a `.env` file in the root directory.

Example:

```env
API_KEY=your_api_key_here
```

---

## Run the Project

```bash
python main.py
```
